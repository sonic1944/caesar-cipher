# Caesar Cipher
A simple web application that encrypts and decrypts text using the Caesar Cipher algorithm.

## Features

- Encrypt text using Caesar Cipher
- Adjustable shift value
- Simple browser interface

## Technologies

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository
2. Open `src/index.html` in your browser
